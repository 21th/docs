We’ve now covered configuring statecharts and setting up their states, transitions, and options.

Next, we need to learn how to _execute_ our machines in code. Depending on where you’re using XState, you’ll likely need to run machines differently. We’ll start with the VanillaJS approach for now.

## Interpret

The `interpret` interpreter provides us with a “running” and _interactive_ version of our machine.

```ts twoslash
import { createMachine, interpret } from "xstate";

const machine = createMachine({});

const service = interpret(machine).start();
```

In XState, we call that “running” machine a `service`. The most common way to interact with a service is to send it events:

{/* TODO: maybe refer to this as an 'actor' instead? \ */}

```ts twoslash
import { createMachine, interpret } from "xstate";

const machine = createMachine(
  {
    on: {
      SAY_HELLO: {
        actions: "sayHello",
      },
    },
  },
  {
    actions: {
      sayHello: () => {
        console.log("Hello!");
      },
    },
  },
);

const service = interpret(machine).start();

service.send({
  type: "SAY_HELLO",
});

// Logs 'Hello!'
```

You can also subscribe to the service via `service.subscribe`:

```ts twoslash
import { createMachine, interpret } from "xstate";

const machine = createMachine(
  {
    on: {
      SAY_HELLO: {
        actions: "sayHello",
      },
    },
  },
  {
    actions: {
      sayHello: () => {
        console.log("Hello!");
      },
    },
  },
);

const service = interpret(machine).start();

// ---cut---

// Fires whenever the state changes
const { unsubscribe } = service.subscribe((state) => {
  console.log(
    // We’ll learn what goes inside here soon!
    state,
  );
});
```

You can also stop the service by running `service.stop()` which cleans up the service and runs any relevant exit actions:

```ts twoslash
import { createMachine, interpret } from "xstate";

const machine = createMachine(
  {
    on: {
      SAY_HELLO: {
        actions: "sayHello",
      },
    },
  },
  {
    actions: {
      sayHello: () => {
        console.log("Hello!");
      },
    },
  },
);

const service = interpret(machine).start();

// ---cut---

service.stop();
```
